Techno Web Clone (Bootstrap + HTML + CSS)
📌 Project Overview

This project is a Techno website clone built using HTML, CSS, and Bootstrap. It replicates the layout and design of a modern tech company website, focusing on responsiveness, clean UI, and reusable components.

The goal of this project is to practice front-end development skills, especially working with Bootstrap grid systems, components, and responsive design techniques.

🚀 Features

>Responsive design for all devices (mobile, tablet, desktop)

>Modern UI layout inspired by techno/IT service websites

>Bootstrap-based grid system

>Navigation bar with smooth scrolling

>Hero section with call-to-action buttons

>Services section with cards

>About section

>Contact form UI

>Footer with social links
